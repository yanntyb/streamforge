2. Information We Collect

We collect the following categories of information:

a) Via TikTok Content Posting API

StreamForge uses TikTok's official Content Posting API to publish video content to our TikTok account. Through this API integration, we access and store the following data:

  • TikTok OAuth access tokens and refresh tokens for our operator account.
  • TikTok Creator ID associated with the @streamforge account.
  • Account-level information returned by TikTok's Query Creator Info endpoint (username, follower count, available scopes).
  • Post status information returned by TikTok's video upload status endpoints.

Important: StreamForge's use of the TikTok Content Posting API is limited to managing our own @streamforge creator account. We do not collect, process, or store any personal data from TikTok viewers, followers, or third-party users through the API.

b) Via This Website

This website (yanntyb.github.io/streamforge) does not use cookies, analytics scripts, tracking pixels, or any form of visitor tracking. No personal data is collected from visitors to this website. The site is hosted on GitHub Pages, which may collect standard server logs (IP address, browser type, referring page) as described in GitHub's Privacy Statement.

c) Contact Inquiries

If you contact us by email, we collect your email address and the contents of your message solely to respond to your inquiry. This information is not shared with third parties and is deleted once the inquiry is resolved.

3. How We Use Information

Information collected is used solely for the following purposes:

  • Authenticating with TikTok's Content Posting API to publish video content on @streamforge.
  • Monitoring the upload and publication status of posted content.
  • Responding to contact or content removal requests submitted by users or rights holders.

We do not use collected information for advertising, profiling, or any purpose beyond those listed above. We do not sell, rent, or trade any data to third parties.

4. TikTok API Data — Specific Disclosures

StreamForge's application integrates with TikTok's Content Posting API for the sole purpose of programmatically posting video content to our own @streamforge TikTok creator account. The following disclosures apply specifically to data obtained via the TikTok API:

  • Scope of access: StreamForge requests only the API scopes necessary for video upload and publication. We do not request access to followers' data, private messages, or any data beyond what is required for content posting.
  • Token storage: TikTok OAuth access tokens and refresh tokens are stored server-side in our application database with encryption at rest. They are never exposed in client-side code, logs, or public repositories.
  • Permitted use only: Data obtained via the TikTok API is used exclusively for its stated purpose — posting video content on StreamForge's account. It is not used for analytics, advertising, or any secondary purpose.
  • No third-party sharing: TikTok API data is not shared with any third party, sold, or used for cross-platform tracking.
  • Compliance: Our use of the TikTok API complies with TikTok's API Terms of Service and TikTok's platform policies.

5. Data Retention

TikTok API access tokens are retained only for as long as they are needed to operate the @streamforge account. Tokens are invalidated and deleted when:

  • The @streamforge TikTok account is disconnected from our application.
  • Tokens expire and are not renewed.
  • The StreamForge service is discontinued.

Contact inquiry data (email addresses) is deleted within 30 days of the inquiry being resolved.
